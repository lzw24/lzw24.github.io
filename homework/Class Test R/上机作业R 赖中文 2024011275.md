### 2. 按Species列将数据分成3组，分别计算Sepal.Length的均值和标准差，保存为一个csv文件，提供代码和csv文件的内容。

```R
# 分别计算均值
> mean_Length <- aggregate(Sepal.Length ~ Species, iris, mean)

# 分别计算标准差
> sd_length <- aggregate(Sepal.Length ~ Species, iris, sd)

# 存入 result 这一变量中，Sepcies 列采用 mean_Length 变量中的 Species 列
> result <- data.frame(Species = mean_Length$Species,Mean = mean_Length$Sepal.Length,SD = sd_length$Sepal.Length)

# 查看 result 变量
> print(result)
     Species  Mean        SD
1     setosa 5.006 0.3524897
2 versicolor 5.936 0.5161711
3  virginica 6.588 0.6358796

# 修改列名
> colnames(result) <- c("Species","Mean.Sepal.Length","SD.Sepal.Length")

# 以 csv 文件导出
> write.csv(result, "C:\Users\63271\Desktop\生物信息学\作业\上机作业R\iris.Sepal.Length.Summary")
```

### 3. 对不同Species的Sepal.Width进行One way ANOVA分析，提供代码和输出的结果。

```R
# 对 Species 和 Sepal.Width 进行 ANOVA，检验上述 setosa ，versicolor ，virginica 三组在 Sepal.Width 层面有无差异

> summary(aov(Sepal.Width ~ Species , data = iris))
             Df Sum Sq Mean Sq F value Pr(>F)    
Species       2  11.35   5.672   49.16 <2e-16 ***   # 组间差异 SSb
Residuals   147  16.96   0.115                      # 组内差异 SSw 
---
Signif. codes:  0 ‘***’ 0.001 ‘**’ 0.01 ‘*’ 0.05 ‘.’ 0.1 ‘ ’ 1
```

从上述结果可知，由于 p 值远小于 0.001 ，可认为：至少有两个不同的 Species ，它们的 Sepal.Width 存在统计学上的显著差异。但此处总体的方差分析无法表明哪两组存在明显差异。可以对某两种 Species 分别进行进一步的统计学检验，探究相关性。
